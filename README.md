# termux3.0

Termux custom script for zshrc , bashrc and welcome page 

## one-line command for installation 
 (open Termux ,copy all command given below ) 
```
pkg install git
git clone https://github.com/keralahacker/termux3.0  
cd termux3.0 
chmod +x install.sh 
./install.sh
```
